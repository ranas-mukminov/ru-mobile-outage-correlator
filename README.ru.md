# ru-mobile-outage-correlator

SRE-инструмент, помогающий соотнести внутренние инциденты с публичными и собственными сигналами о сбоях российских операторов связи (МТС, Мегафон, Tele2, Билайн, Ростелеком, Yota и др.). Проект создан для легальной и безопасной корреляции: он **не** предназначен для обхода блокировок или атак на сети.

## Возможности
- сбор внешних сигналов (статус-страницы, новости, собственные пробы по регионам);
- импорт внутренних инцидентов из логов/метрик;
- вычисление вероятности, что инцидент вызван внешним операторским сбоем;
- генерация коротких русских резюме для постмортемов и статус-страниц.

Лицензия: Apache-2.0, подходит для коммерческого использования и консалтинга.

## Быстрый старт
1. Установите зависимости: `pip install -e .`.
2. Подготовьте синтетические данные:
   - `examples/demo_small_saas/external_signals.json`
   - `examples/demo_small_saas/internal_incidents.json`
3. Запустите корреляцию:
   ```bash
   ru-mobile-outage-correlator correlate \
     examples/demo_small_saas/internal_incidents.json \
     examples/demo_small_saas/external_signals.json \
     --output examples/demo_small_saas/report.json
   ru-mobile-outage-correlator report examples/demo_small_saas/report.json
   ```

## Интеграция в observability/SRE
- Данные из Loki/ELK экспортируйте в JSON и подайте в `collect-internal`.
- Метрики Prometheus агрегируйте и выгружайте в файл для офлайн анализа.
- Внешние сигналы подключайте только через публичные API/фиды, соблюдая robots.txt и ToS.

## AI-интеграция
Модуль `ai/summaries.py` задаёт интерфейс для генерации текстов. По умолчанию используются локальные шаблоны. В `docs/ai_integration.md` показано, как подключить внешнюю LLM без ключей в репозитории. Не отправляйте чувствительные данные третьим сторонам без договоров.

## Услуги и поддержка
За внедрением, адаптацией под российские реалии и обучением команд обращайтесь: **https://run-as-daemon.ru**. Доступны услуги по настройке SRE/observability-стека, корреляции инцидентов с внешними авариями и обучению инженеров.
